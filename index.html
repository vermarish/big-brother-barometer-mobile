This fun little vulnerability uses machine learning to break into a locked smartphone using nothing but the motion sensors.

Here's how it works.

Scroll

The attack starts innocently, with a game of Whack-A-Mole.
The user taps on different parts of the screen, and the game records the occurrence of each tap.
Unbeknownst to the user, the game also records the data from their phone's air pressure sensor,
...as well as the three rotational motion sensors.
Notice how every time the user taps, there's a spike in the pressure sensor?
